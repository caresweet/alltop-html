## 二、托管方案二选一

### 方案 A：OSS 静态托管（推荐，最省钱）

**适合：** 纯展示型官网（当前网站就是这种），无后台、无数据库。  
**费用：** OSS 约 3–10 元/月（按存储+流量计费）+ CDN 约 5 元/月 ≈ **全年 100 元左右**。  
**优点：** 极便宜、打开快、免维护、不怕宕机。  
**缺点：** 将来想加"会员系统、在线下单"等动态功能需另搭服务器。

### 方案 B：轻量应用服务器（灵活，可扩展）

**适合：** 将来可能加"在线咨询后台、案例管理、员工登录"等动态功能。  
**费用：** 2 核 2G 3M 带宽，新人价 **约 108 元/年**（续费约 268 元/年）。  
**优点：** 完整服务器，想装什么装什么，可扩展。  
**缺点：** 要自己维护，比 OSS 贵一点。

**建议：** 现阶段选 **方案 A（OSS）**，省钱省心；将来要加功能再升级到方案 B，网站文件不用改。

## 四、部署步骤（备案通过后）

### 方案 A 部署（OSS）

```
1. 登录阿里云控制台 → 对象存储 OSS → 创建 Bucket
   - 名称：zhongtuo-website（自定义）
   - 区域：华东 2（上海）
   - 读写权限：公共读
   - ✅ 开启「静态页面」- 默认首页填 index.html

2. 上传文件
   - 把本地 zhongtuo-website/ 目录下的所有文件上传到 Bucket
   - 包含：index.html 和 assets/ 文件夹（含 logo.png）

3. 绑定域名
   - Bucket → 传输管理 → 域名管理 → 绑定你的域名
   - 自动添加 CNAME 解析

4. 开启 CDN 加速（可选，推荐）
   - CDN 控制台 → 添加域名 → 填 OSS 的域名
   - 加速后全国访问更快

5. 配置 HTTPS（可选，推荐）
   - SSL 证书控制台 → 申请免费 DV 证书 → 绑定域名
   - 在 OSS/CDN 里开启 HTTPS
```

### 方案 B 部署（轻量服务器）

```
1. 登录轻量应用服务器控制台 → 找到服务器 → 重置密码

2. 用 SSH 或宝塔面板连接服务器
   - 安装宝塔面板：https://www.bt.cn/ （可视化操作，推荐）
   - 或命令行装 Nginx

3. 上传网站文件到 /www/wwwroot/zhongtuo/

4. 宝塔 → 网站 → 添加站点 → 域名填你的域名 → 根目录选 /www/wwwroot/zhongtuo

5. 域名解析
   - 阿里云域名控制台 → 解析 → 添加 A 记录
   - 记录值填服务器公网 IP

6. 配置 SSL 证书（宝塔一键申请免费证书）
```
